# piezas-html

Un ejemplo de una pieza-html (componente)
seria el siguiente header el cual se compone
de otros elementos html en su interior

```
<header>
    <img alt="logo" src="./assets/logo.png" class="logo">
    <div class="linea-dorada"></div>
    <h1 class="serif">Mercanc&iacute;a<span class="span">.Store</span></h1>
    <p class="mayusculas">amazon affiliate program</p>
</header>
```

## uso 

El header debe de estar colocado
entre ```<body> y </body>```

y los archivos .css que contengan las clases
necesarias especificadas en los elementos
html con el atributo class deberan de estar
ubicados entre ```<head> y </head>```
